# 3D File Tools

Browser-based 3D file viewer, converter, and repair for 3D printing community.

## Why This Wins
- **Zero competition** — no good free online tool exists
- **Specific unmet needs**: "stl viewer online with measurement" / "with dimensions" in autocomplete
- **3D printing market growing 20%+ YoY** — expanding audience
- **Passionate community** that shares tools on Reddit (r/3Dprinting = 2M+ members)
- **Rust is ideal** for mesh processing (three-d, nalgebra crates)

## Revenue Model
- **Primary**: Display ads ($10-15 RPM — tech/hobby niche)
- **Secondary**: Affiliate links to 3D printers and filament
- **Volume**: Smaller market but zero competition = easier ranking

## Tools to Build
1. STL viewer (with measurement/dimensions — the unmet need)

6. GLTF/GLB viewer
7. Mesh repair (fix non-manifold, holes)
